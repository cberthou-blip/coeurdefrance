# Cœur de France

Site statique de présentation de Cœur de France, livre blanc politique libre, sans parti ni affiliation.

## Contenu

- `index.html` : page publique principale
- `styles.css` : styles mobile-first
- `server.mjs` : serveur local statique
- `corpus_coeur_de_france/` : corpus de travail nettoyé et exploitable

## Lancer en local

```bash
node server.mjs 4173
```

Puis ouvrir `http://127.0.0.1:4173/`.
